# Student Grade Management System (Python)

##  Description
This is a simple Python console-based project that allows users to:
- Add student grades
- Update student grades
- Delete student records
- View all students and their grades

##  Technologies Used
- Python
- Dictionary
- Functions
- Loops

##  How to Run
1. Install Python
2. Run the file:
   python student_grades.py

##  Features
- Menu-driven program
- Easy to use
- Beginner-friendly project

##  Use Case
Suitable for Python beginners and students learning basic CRUD operations.
